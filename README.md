## Setup Instructions

### Prerequisites

- Node.js (v18 or higher)
- npm (v9 or higher)

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Using Code Suggestions Feature

The code suggestions feature requires Ollama to be running locally:

1. Install Ollama from [https://ollama.ai/](https://ollama.ai/)
2. Run the Ollama service:
   ```bash
   ollama serve
   ```
3. Pull the CodeLlama model:
   ```bash
   ollama pull codellama
   ```

Once Ollama is running with the CodeLlama model, the code suggestions feature will work in the editor.
